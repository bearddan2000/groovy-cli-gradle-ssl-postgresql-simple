# groovy-cli-gradle-ssl-postgresql-simple

## Description
A groovy gradle build, that connects to postgresql database.

Uses self-sign ssl.

## Tech stack
- groovy
- gradle
  - 6.8.2

## Docker stack
- alpine:edge
- postgresql:alpine
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
